# XP Sound Files

Place the following MP3 files here for authentic XP sounds:
- startup.mp3  — Windows XP startup sound
- logon.mp3    — Windows XP logon sound
- chord.mp3    — Windows XP chord (error/notify)
- close.mp3    — Window close sound

These can be found from various free XP sound archives online.
The app gracefully handles missing sounds.
